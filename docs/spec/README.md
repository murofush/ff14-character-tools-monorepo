# 仕様書インデックス

このディレクトリ配下の仕様書が仕様のSSOT（唯一の真実）です。  
ただし、**Confirmed** と明記された仕様のみがSSOTとして有効です。

## 運用ルール

- 仕様が確定したら必ず仕様書へ反映する
- 新規仕様は `docs/spec/_template.md` の書式を使用する
- 仕様変更時は該当仕様書とこの索引を両方更新する
- ファイル名は内容ベースとし、番号の連番は付けない（履歴はGitで管理する）
