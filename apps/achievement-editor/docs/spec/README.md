# 仕様書インデックス（SSOT）

## 運用ルール

- 仕様変更時は、該当仕様書と本インデックスを同時更新する。
- 仕様書ファイル名は内容ベース（連番禁止）。
- 新規仕様は `docs/spec/_template.md` を使用する。
